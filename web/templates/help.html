Hilfe & Anleitung

Willkommen beim TravelWifi-System! Hier finden Sie wichtige Informationen und Tipps zur Nutzung Ihres Geräts.

Bedeutung der Status-LEDs

Die LEDs auf Ihrem Gerät geben Auskunft über den aktuellen Status:

  • **Grün blinkend:** Das System ist hochgefahren und wartet auf eine Verbindung mit einem externen Netzwerk.
  • **Grün leuchtend:** Eine Internetverbindung wurde erfolgreich hergestellt. Das System ist betriebsbereit.
  • **Rot blinkend:** Ein kritischer Fehler ist aufgetreten oder es konnte keine Verbindung zum Internet hergestellt werden. Überprüfen Sie die Diagnose-Seite für Details.

Grundlegende Fehlersuche

Wenn die Internetverbindung nicht funktioniert, versuchen Sie bitte folgende Schritte:

  1. Stellen Sie sicher, dass Ihr Gerät sich in Reichweite eines bekannten Netzwerks befindet.
  2. Führen Sie auf der Startseite einen neuen Netzwerk-Scan durch.
  3. Prüfen Sie auf der Diagnose-Seite den System-Log auf Fehlermeldungen.
  4. Wenn das Problem weiterhin besteht, versuchen Sie einen Neustart des Geräts.

Hinweise zur optimalen Nutzung

  • **Standort:** Platzieren Sie das Gerät so frei wie möglich, um eine optimale WLAN-Reichweite und Signalstärke zu erzielen.
  • **Werksreset:** Sollte das System in einen unbrauchbaren Zustand geraten, nutzen Sie die Werksreset-Option in den Einstellungen, um es auf seine ursprünglichen Werte zurückzusetzen.
